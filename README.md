## 📥 Comment installer ?

1. Télécharge le launcher **SuperNova**
2. Lance le .exe
3. Choisis ton modpack
4. Clique sur **JOUER** !

## 🛠️ Ajouter un mod

Les administrateurs peuvent ajouter des mods en :
1. Uploadant le `.jar` dans `modpacks/<nom>/mods/`
2. Mettant à jour `modpacks.json`
